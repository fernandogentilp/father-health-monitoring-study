# father-health-monitoring-study
Basicamente esse é um estudo de caso, que usei análise de dados, com plotagem de gráficos e um pequeno treinamento de modelo baseado em Regressão Linear, para estudo de caso das percentagens da saúde do meu pai, usando seus exames como base para essas análises. 
 Como sou totalmente iniciante nesse assunto de IA/Análise de Dados, usei LLM's para me auxiliar no estudo de caso, e esse estudo está aberto para modificações e atualizações a medida que vou aprendendo mais sobre o assunto de Aprendizado de Máquina e a medida que os exames do meu pai forem atualizando. É praticamente um projeto brinquedo que estou fazendo durante as férias.
